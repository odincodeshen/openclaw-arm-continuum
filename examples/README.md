# Examples

These examples are meant to be copied directly into Telegram after OpenClaw is
running.

Start here:

```text
examples/telegram-commands/smoke-test.txt
```

Then try the focused command sets:

- `examples/telegram-commands/memory-rag.txt`
- `examples/telegram-commands/document-rag.txt`
- `examples/telegram-commands/web-search.txt`
- `examples/telegram-commands/cron-briefings.txt`
- `examples/telegram-commands/voice-notes.txt`
- `examples/telegram-commands/demo-profile.txt`

Most files contain only Telegram messages that can be sent as-is. The document
RAG flow requires either a public Google Docs URL or a Telegram file upload, so
use `document-rag.txt` after replacing the sample URL with your own public
document.

The examples do not contain tokens, chat IDs, hostnames, IP addresses, or
personal data.
